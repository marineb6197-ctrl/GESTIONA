Envoi depuis votre messagerie

Aucune configuration technique n’est nécessaire.

GESTIONA prépare automatiquement le destinataire, l’objet et le récapitulatif. Vous choisissez ensuite votre messagerie et vous cliquez sur Envoyer depuis votre propre boîte mail.

Pour joindre un PDF, utilisez d’abord le bouton « Imprimer / PDF », enregistrez le document, puis joignez-le dans votre messagerie.

Préparer la commande
